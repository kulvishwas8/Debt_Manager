  
DebtManager is a streamlined personal finance tool designed to help users track,visulize, and optimize their debt repayment strategies. Whether you're juggling credit cards, student loans, or personal borrowings, DebtManager provides clarity and control over your financial obligations.

## Features

- Debt Tracking: Add multiple debts with details like principal, interest rate, due date, and minimum payment.
- Lumpsum Payment: Choose to pay lumpsum and optimize your payoff strategy.
- Progress Visualization: Interactive charts showing total debt, interest paid, and estimated payoff timeline.
- Secure & Private: We do not store any user data
##  Tech Stack

- Frontend: Streamlit (for rapid UI development)
- Backend: Python (Pandas, NumPy for calculations)
  

## Installation

```bash
git clone https://github.com/kulvishwas8/debtmanager.git
cd debtmanager
pip install -r requirements.txt
streamlit run app.py
```

##  Usage

1. Launch the app using Streamlit.
2. Add your debts with relevant details.
3. Monitor progress and adjust as needed.
4. Export reports or share insights with a financial advisor.
